## Decisions

| Decision | Rationale |
|----------|-----------|
| C# / .NET 8 | Enterprise quality + free step-debugging via VS debugger |
| Core + Runner split | Engine reusable; GUI = second frontend to same library |
| YAML for pipeline definitions | Familiar to GitLab/GitHub/Azure users |
| System.CommandLine for CLI | Modern .NET CLI framework, subcommands, help generation |
| YamlDotNet for parsing | Standard .NET YAML library |
| Templates as C# code, not YAML files | Composable via Compose(), type-safe, IDE support |
| Prometheus campaign for Claude Code | Strategic planning here, execution there |
